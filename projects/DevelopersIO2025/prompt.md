```
## 概要
"Hapio AI"は、AIがあなたのプレゼンテーションを多角的に分析・評価し、具体的な改善提案を行うプレゼンテーション練習プラットフォームです。

## 主な機能
### シチュエーション設定
あなたの想定するプレゼンテーションのシチュエーションを選択することで、あなたのプレゼンテーションをAIが評価してくれます。
以下の項目を詳細に設定できます。

- Topic - 発表テーマ
- Scene - 発表場面
- Audience Type - 聴衆の種類
- Formality Level - フォーマル度
- Language - 使用言語
- Presentation Purpose - 発表目的


選択したシチュエーションにそったプレゼンテーションが実施されているかをHapio AIは評価します。

また、制限時間も設定することができ、ユーザーの時間管理も行なってくれます。

### 多角的評価システム
ユーザーが行ったプレゼンテーションは音声から文字起こしされ、顔の表情なども分析します。
シチュエーション、スライド資料、プレゼン内容（文字起こし）、顔の表情、プレゼンにかかった時間などをもとに、総合的にユーザーのプレゼンテーションを評価します。

以下の5つの観点から総合的にプレゼンテーションを評価し、点数を算出します。
- Content Accuracy - 内容の正確性・適切性
- Language & Wording - 言葉遣い・語彙選択の適切性
- Situation Appropriateness - シチュエーションとの整合性
- Facial Expression - 表情・非言語コミュニケーション
- Structure & Logic	構成・論理展開の明確性

これらを総合的に踏まえ点数を算出します。
資料のスライド単位でHapio AIは評価してくれるとともに、最終的にはプレゼン全体を通した総合的な評価を実施してくれます。
ユーザーのプレゼン内容を踏まえ、Hapio AIはより良いプレゼン内容を自動で提案します。
また、過去のプレゼンの評価を見ることも可能です。

## 解決したい課題
このような悩みはありませんか？
- 人前で説得力のある質の高いプレゼンを行いたい

- 発表前にプレゼンテーションの練習をしたい
```

```
/src/app
├─ _client                         // クライアントサイドのコード
│  ├─ common/                      // 共通で使用されるコンポーネント・テキストト

   │   ├── core/          # コアドメインモデル
   │   │   └── */model/  # ドメインモデル
   │   │   └── */domain-function/  # ドメインサービス
   │   │   └── */repository/  # リポジトリ
   │   └── master-data/       # マスターデータなどのモデルなど
   │   └── support/       # APIクライアントなどサポート機能

   │  ├─ projects/                 // プロジェクト一覧
```

## 開発手法
- コーディングエディタ：Cursor
- AI Agent：Cline

AI Agentを使った開発を行うことで、業務が終わった後などのプライベートの時間でも開発を行うことができました。
Clineに全部賭けろという記事がでたばっかりの時かつ、Claude3の出現により「今しかない」という気持ちで開発を行いました。
やく4ヶ月で1つのアプリをリリースすることができました。

Clineは従量課金でAPIを呼び出すため、計算したら4ヶ月で7万円ほどかかっていた

## ポイント
個人開発だからといってアーキテクチャを疎かにしないことが重要です。
クリーンアーキテクチャを採用し、Eslintで依存関係の管理を行うことで、コードの品質を保ちながら開発を行うことができました。

AIエージェントが豊富な今、個人開発に挑戦することで商用レベルのアプリをリリースすることができると思います。
